# MailDrag Pro 📧

MailDrag Pro is a lightweight, intuitive, and fully responsive drag-and-drop email template builder. Built with Vanilla JavaScript and Tailwind CSS, it allows users to visually assemble, customize, and export professional HTML email campaigns without writing a single line of code.

## ✨ Features

* **Drag & Drop Interface:** Easily drag blocks (typography, media, marketing elements, structural dividers) directly onto the canvas.
* **Pre-built Templates:** Jumpstart your campaigns with ready-to-use templates for E-commerce Promos, Newsletters, Welcome Series, and Event Invitations.
* **Real-time Preview:** Toggle between desktop and mobile viewport modes to ensure your emails look great on any device.
* **Global Themes:** Instantly change the look and feel of your email with one-click global themes (Light, Dark, Playful, Elegant, Corporate).
* **Block Customization:** Click on any block to open the settings panel and tweak text, colors, alignment, and more.
* **History Management:** Built-in Undo and Redo functionality (supports keyboard shortcuts `Ctrl+Z` / `Ctrl+Y`).
* **Export to HTML:** Export your finished design into clean, copy-pasteable HTML, or download it directly as an `.html` file ready for your email service provider (Mailchimp, SendGrid, etc.).

## 🛠️ Built With

This project requires no complex build tools or package managers. It runs directly in the browser.

* **HTML5 & CSS3**
* **Vanilla JavaScript** (No heavy frameworks)
* [Tailwind CSS](https://tailwindcss.com/) (Loaded via CDN for rapid UI styling)
* [Lucide Icons](https://lucide.dev/) (Loaded via CDN for crisp, clean SVG icons)

## 🚀 Getting Started

Because MailDrag Pro is built with standard web technologies and CDNs, setting it up is entirely frictionless.

### Prerequisites
* A modern web browser (Chrome, Firefox, Safari, Edge).
* An active internet connection (to load the Tailwind and Lucide CDNs).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/maildrag-pro.git](https://github.com/yourusername/maildrag-pro.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd maildrag-pro
    ```
3.  **Run the application:**
    Simply open the `index.html` file in your preferred web browser. 
    *(Alternatively, you can use a local server like VS Code's "Live Server" extension for a better development experience).*

## 💡 Usage

1.  **Choose a Starting Point:** When the app loads, choose to start from a blank canvas or select one of the pre-made templates.
2.  **Build your Layout:** Open the accordions in the left sidebar (Blocks tab) and drag elements like Headings, Images, Articles, or Buttons onto the central canvas.
3.  **Customize:** Click on any block you dropped onto the canvas. The left sidebar will switch to the "Settings" tab, allowing you to edit the content and styling of that specific element.
4.  **Preview:** Use the Desktop/Mobile icons at the top right to verify your layout is responsive.
5.  **Export:** Click the "Export" button in the top right corner. You can either copy the generated HTML code to your clipboard or download it as an `.html` file.
